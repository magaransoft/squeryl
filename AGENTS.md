# AGENTS.md

## Writing Style

When writing or editing `AGENTS.md`, `CLAUDE.md`, memory files, or agent-notes, use the "caveman ultra" writing grammar defined in [`ai/caveman.md`](ai/caveman.md). This applies only to those files. Session logs (chat transcripts and agent-notes under `session-logs/`) are not affected and should be written in normal prose.
